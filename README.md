# Products API Tests

This project contains automated API tests for the Products API using Playwright.

Test API URL: https://api.practicesoftwaretesting.com/api/documentation#/Product/storeProduct

## Project Structure

- `tests/` - Contains Playwright test specs for brands and products.
- `helpers/` - Helper utilities for REST API interactions.
- `playwright.config.ts` - Playwright configuration file.
- `docker-compose.yml` - Docker Compose setup for running tests in a containerized environment.
- `playwright-report/` - Test reports generated by Playwright.
- `test-results/` - Raw test results.

## Prerequisites

- [Docker](https://www.docker.com/get-started) installed on your machine.
- [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed (for running tests locally).

## Running Tests with Docker

To run the Playwright tests using Docker Compose, use the following command:

```sh
docker-compose up --build
```

This will build the Docker image (if needed) and execute the tests inside a container.

## Running Tests from Local Terminal

To run the Playwright tests directly from your local terminal:

1. Install dependencies (if not already done):
   ```sh
   npm install
   ```
2. Run all tests:
   ```sh
   npx playwright test
   ```
3. Run a single test suite (e.g., only `brands.spec.ts`):
   ```sh
   npx playwright test tests/brands.spec.ts
   ```
4. Run a single test case (by test name):
   ```sh
   npx playwright test -g "test case name"
   ```

## Viewing Reports

After the tests complete, you can view the Playwright HTML report in the `playwright-report/index.html` file.
